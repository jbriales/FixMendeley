(Ongoing work)

Python package to clean and fix conflicting Mendeley entries  
- fuse_entries.py gathers information of repeated documents  
- import_dblp.py grabs information of dblp database to complete fields

Mendeley local SQLite database is handled via [peewee](https://github.com/coleifer/peewee)
NOTE: You may need to manually create a symlink to jesusbriales@uma.es@www.mendeley.com.sqlite
